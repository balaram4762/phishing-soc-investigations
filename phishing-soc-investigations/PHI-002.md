# SOC Investigation Ticket — PHI-002

Case ID: PHI-002
Date: 2026-06-29
Severity: HIGH
Attack Type: Credential Phishing

## What this email/site was about
This one was impersonating Carrefour, the supermarket chain. The fake domain was carrefour.negocie-aqui.com — attackers added "negocie-aqui" (which means "negotiate here" in Portuguese) to make it look like a legit Carrefour promotion page.

## What I did and found
On VirusTotal, 1 out of 92 vendors flagged the URL as malicious so far. The site is hosted on Amazon AWS in the US, which is something attackers do because AWS is a trusted, widely-used service — it makes the site look more legitimate and harder to block at a network level. I also noticed VirusTotal flagged 4 known malicious files that had connected to this same IP address before, which tells me this IP has been used for bad activity multiple times, not just this one phishing page.

## IOCs found
- Domain: carrefour.negocie-aqui.com
- URL: https://carrefour.negocie-aqui.com/
- IP: 13.32.205.46

## What I'd recommend
Block the domain at the firewall and report it. Since the IP has a history of hosting malicious files, it's worth flagging the whole IP range for monitoring too.
